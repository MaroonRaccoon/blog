An theorem

Here we prove an theorem. Wag!!

Let $n\in\mathbb{N}$. There is exactly one $n$-element subgroup of $SO(2).$ Let $G_n$ be an $n$-element subgroup of $SO(2)$, and define $$H_n = \left\{ \mathrm{exp}\left( {i\cdot\frac{2\pi k}{n}} \right) : k\in\mathbb{Z} \right\}.$$ We will prove that $G_n = H_n$.

Suppose $z\in G_n$. Then, by Lagrange's theorem, we know that $z^n = 1$. From this, we find $|z|=1$, and taking $\theta=\mathrm{arg}(z)$, we have $n\theta = 2\pi k$ for some $k\in\mathbb{Z}$, i.e. $\theta = \displaystyle\frac{2\pi k}{n}$ for some $k\in\mathbb{Z}$. Hence $z\in H_n$, so $G_n \subseteq H_n$. But $|H_n| = n = |G_n|$, and so $G_n = H_n$, completing the proof.
